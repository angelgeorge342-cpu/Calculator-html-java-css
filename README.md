# Calculator Web App

## Description
This project is a web-based calculator built using HTML, CSS, and JavaScript.  
It allows users to perform basic arithmetic operations through a simple and interactive user interface.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Addition, subtraction, multiplication, and division
- Interactive button-based input
- Clear/reset functionality
- Responsive layout for different screen sizes

## How to Run
1. Download or clone the repository
2. Open the `index.html` file in any modern web browser
